# 🎾 Tennis Game Analytics

##  Project Overview

This project analyzes tennis player data to generate insights on **rankings, performance, and point distribution**.

It provides a structured dashboard to visualize:

* Player rankings
* Leaderboard comparison
* Top players by points
* Performance insights

The goal is to convert raw tennis data into **clear, data-driven insights** for better understanding of player performance.

##  Objectives

* Analyze player rankings and trends
* Identify top-performing players
* Compare competitors across regions
* Understand point distribution patterns
* Build interactive visual dashboards

##  Features

### 🔹 Leaderboard

Displays top-ranked players with their points.

### 🔹 Filtered Competitor

Compare selected players based on:

* Country
* Rank
* Points

### 🔹 Top Players by Points

Bar chart showing **Top 10 players** by total points.

### 🔹 Insights

* Geographic Dominance
* Ranking Volatility
* Performance Concentration
* Category Data Density

##  Tech Stack

* **Python** (Pandas, NumPy)
* **Visualization**: Matplotlib / Seaborn
* **Dashboard**: Streamlit / Power BI
* **Tools**: Jupyter Notebook, VS Code, PyCharm

##  How to Run?

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/shreyathaware123/Game-Analytics-Unlocking-Tennis-Data-with-Sportradar-API
cd Tennis-Game-Analytics
```

---

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 3️⃣ Run the Project

#### ▶️ If using Streamlit Dashboard:

```bash
streamlit run app.py

```


##  Key Insights

* Top 10 players contribute a large share of total points
* Rankings after top 50 show higher volatility
* Significant performance gap exists between top and mid-tier players
* Certain regions dominate global rankings
